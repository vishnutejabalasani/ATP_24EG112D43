
# Week 1 Assignments - JavaScript Basics

This folder contains JavaScript programs that help in learning the basic concepts of programming. Each file focuses on a specific task.

### Day 1: Basic Logic and Math
* **day1ass1.js**: Finds which number is bigger between two numbers.  
* **day1ass2.js**: Finds the biggest number among three different numbers.  
* **day1ass3.js**: Adds up all the numbers in a list (array) and shows the total sum.  
* **day1ass4.js**: Finds the smallest number in a list (array).  
* **day1ass5.js**: A reusable function that takes three numbers and tells you which one is the biggest.  
* **day1ass6.js**: A reusable function that takes a list of numbers and returns their total sum.  
* **day1ass7.js**: A reusable function that looks for a specific number in a list and tells you where it is located (its position).  

### Day 2: Working with Data (Objects and Arrays)
* **day2ass1.js**: Shows how to manage a list of employees. It includes:  
  * Adding a new employee to the list.  
  * Removing an employee by their name.  
  * Changing specific details (like marks) for an employee.  

## Concepts Learned

1. **Variables**: Using `let` and `const` to store information like numbers and names.  
2. **Conditions (If/Else)**: Making the computer choose between different options based on rules.  
3. **Loops (For Loop)**: Telling the computer to do the same task many times, like looking through every item in a list.  
4. **Arrays**: Storing a group of items (like a list of marks or numbers) in a single place.  
5. **Functions**: Writing a block of code once and using it many times with different inputs.  
6. **Objects**: Storing detailed information about a single thing (like an employee with a name, ID, and marks).  
7. **Array Methods**: Using special built-in tools like `splice`, `find`, and `findIndex` to easily change or search through lists.
