# Week 6 Assignments

This week was all about getting into React and then putting everything together to build a full MERN stack application.

### Day 15: Starting with React
I started by learning the basics of React. I built a simple app to practice how components work and how to pass data between them using props. 

Key things I worked on:
- Creating basic components like Navbar and Footer.
- Using state with a simple Counter.
- Handling forms to get user input.
- Fetching data from an external API and displaying it on the page.
- Understanding how to list items like Users or Products using props.

### Day 16 & 17: Building a MERN Mini App
The second half of the week was more intense because I built a full-stack application for managing employee records. This involved connecting the frontend and the backend.

Backend side:
- Set up a Node.js and Express server.
- Connected to a MongoDB database.
- Created a schema for employees (name, role, etc.).
- Wrote API routes to handle adding, viewing, updating, and deleting employees.

Frontend side:
- Used React Router to create different pages for the app (Home, Create Employee, Edit Employee).
- Built a dashboard to list all the employees.
- Connected the React forms to the backend API so the data actually saves in the database.

It was a big step up from just doing HTML or simple JavaScript, but it was cool to see how everything connects from the database to the browser screen.
